Menu Bar Commands

A range of options and commands can be accessed via the menu bar at the top of the WAVES Toolbar window. Many of these commands can also be performed by pressing keyboard shortcuts, and additional shortcut commands can be added if needed. The menu bar commands are:

Formula

  • New (Ctrl+Del): Clears the current contents of the Formula List and begins a new math construct. Note that the contents of the Formula List are not saved, so if you want to keep your current math construct, use the Copy command or Copy MathML to Clipboard button to export the math construct to another application (such as Notepad or Word).
  • Import MathML: Opens a file browser window to allow importing of files in the .mathml format. Note that importing a file will clear the current contents of the Formula List without saving them.
  • Undo: Removes the last change made in the Formula List. You can use this command repeatedly to remove several changes.
  • Clear Excerpt (Del): Clears the contents of the currently highlighted line. The line itself is not removed from the List Box; however, since the line is now empty, it is ignored by the MathML renderer and is essentially invisible (this may create MathML errors if the line was part of a multi-line mathematical construct, e.g., a square root or exponent). The line can be reactivated by typing new text into it.
  • Copy to Clipboard (Ctrl+X): Copies the current math construct, formatted as MathML, to the Windows Clipboard for use in other documents or applications.
  • Export as MathML: Opens a file browser window for saving the current math construct as a .mathml file.
  • Export as graphic: Opens a file browser window for saving the current math construct as an image file. The WAVES Toolbar supports exporting to BMP, GIF, JPG, PNG, and TIF formats.

View

  • Always on Top: When checked, this option keeps the WAVES Toolbar window visible even when other applications are active.
  • Hide Menu (F11): This option hides the menu bar. Press the F11 key to unhide the menu bar (if this shortcut is removed, you can also press the Alt key to unhide and navigate the menu bar).
  • Use External MathJax Renderer: This option switches MathML rendering from the included gNumerator renderer to the online MathJax renderer. The MathJax renderer is preferred by some users, but requires an internet connection to perform rendering. The default gNumerator renderer allows the user to render formulas without an internet connection.
  • Show Level 2 Tokens (F2): This option adds additional tokens to the Token Set. The added tokens are the ones designated as Level 2 Tokens in the Edit Master Token List window.
  • Show Token Repository (F3): This option opens a window displaying all available tokens. Use this option when you need a token that isn't present in the default Token Set or the Level 2 Tokens. The main WAVES Toolbar is inactive while the Token Repository is visible, and is re-engaged when the Repository window is closed. The Repository window can be closed using F3 or the Esc key.

Options

  • English Excerpts (Ctrl+E): When checked, this option displays all token identifiers in the Formula List in common English terms (e.g., "fraction" or "square root"). When this option is unchecked, all token identifiers are displayed using MathML tags (e.g., "mfrac" or "msqrt"). This option is checked by default.
  • High Contrast: This option enables a high contrast color mode for the WAVES Toolbar. The high contrast mode may make the WAVES Toolbar interface easier to read for some users.
  • Read Superscript as Power: When this option is checked, superscripted numbers (such as exponentials like x2 and x8) are read as powers (for example, "x squared" or "x to the eighth power"). When this option is unchecked, superscripted numbers are read using the MathML format (for example, "superscript, text x, superscript 2, end superscript"). This option is checked by default.
  • Voice Prompting: This option enables voice prompts whenever you mouse over a token in the Token Set, and also reads the contents of lines in the Formula List when they are clicked on or edited.
  • Edit Keyboard Shortcuts: This option opens a window for customizing keyboard shortcuts for all the commands in the WAVES Toolbar.
  • Edit Master Token List: This option opens a window for editing the availability, levels, and behavior of all available tokens.

Help

  • Index: This option accesses the WAVES Toolbar Help file, which provides basic instructions and tips for using the software.
  • About: This option brings up a window showing information about the WAVES Toolbar, including the current version number and contact information for the developers.
